The files in this directoy are the IBM DC power grid benchmarks. They are
described in the paper:
	Power Grid Analysis Benchmarks
	ASPDAC 2008

The files are names IBMPG<x>.<ext>.bz2.

To uncompress the files, use the Unix/Linux "bunzip2" command.

The file extensions are:
  .gif		a picture of the 2D distribution of voltage the ground grid
  .spice	a SPICE format circuit of the power grid
  .solution	the solution to the SPICE circuit
